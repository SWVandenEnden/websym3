This project is a web interface for symexpress3, cubicequation, quarticequation and sym3taylorserie

Directories:
------------
data           - Contains the user data, see Python module settings.py
favicon        - Favicon files
javascript     - Javascript files (*.js)
style          - Css3 style files (*.css)
template       - Templates
websymexpress3 - Python module web interface

Files:
------
LICENSE          - License file
pylintrc         - Pylint definitions
readme.txt       - This file
websym.py        - Python http server for websymexpress3 (start/main program)
requirements.txt - All the Python packages needed
README.md        - Module information
